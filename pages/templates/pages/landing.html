Tips for Full Utilization

  • Use the description field! When adding a new debtor, add a note about the transaction. It will help you remember the context months later.
  • Record payments immediately. The moment you receive a payment, no matter how small, add it to the system to keep balances accurate.
  • Add new debt clearly. If a debtor takes another item on credit, use the "+ Add Debt" feature. This maintains a perfect log of all transactions, rather than just changing the initial number.
